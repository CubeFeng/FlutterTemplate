Flutter IconFont Builder Demo

如何使用？

点击对应图标, 会自动生成 Flutter Icon 代码;

import 'package:flutter_wallet/generated/icon_font.g.dart';

因svg渲染耗时, 可以使用如下代码进行预缓存;

IconFont.precacheSvg([IconFont.icon]);
